ABOUT

I am a self taught frontend developer based currently in New York City from Chicago, who loves to tackle new challenges. When I am not running around exploring the best desserts in New York, I am learning new tools to improve my coding.
You can find me at meetups like Women Who Code and volunteering with the Anita Borg Intitute at many of their tech events geared at promoting women in STEM fields.
Big lover the outdoors and even bigger lover of books. Currently reading, Notorious RBG.

Process

I make interactive web experiences from using 3 core languages, HTML, CSS and JavaScript. When coding, I think about optimal user experience by making sure the website is responsive and has a mobile design. In addition to cross-browser testing and debugging. I am always striving to use simpler and less code when building websites.

Learning Code

After attending multiple technology conferences and participating in pitch competitions, I decided to learn code with zero programming knowledge. I used a variety of resources from codeacademy, freecodecamp, Udacity, youtube and spent hours exploring many different languages but I really enjoyed designing and coding websites.

ganzorig.com

I coded and designed this fully responsive website from the ground up with HTML,CSS, bootstrap, javaScript and jQuery. Throughout this journey I realized I love designing websites as well. Before this website, I built about 20 websites from scratch and thus gained experience in the best practices for building functional simple websites.
