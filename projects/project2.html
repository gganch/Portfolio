National Society for Bees

One of the sources I used to learn front end development was online classes and one of the classes I took on pluralsight was about the best practices for creating websites and decided to put my practices in use by creating this website about bees. I wanted to create a website around a certain topic so that I can learn about using certain color schemes and layout to evoke emotion and tell a story.

Things that I learned:

- How to select, convert and use videos so that they can successfully integrated into a website
- Properly documenting my code so that it can be read and used more easily by others
- Integrating pieces of code written by other people into my code so that I can provide a better and faster user experience
