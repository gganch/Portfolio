DBOX Agency Website

I was inspired by the website of a marketing agency website which used a simple single page layout to showcase their projects. This website is one of the earlier ones I created and learned a great amount by experimenting with HTML, CSS and bootstrap.

Things that I learned:

- Setting a background image to have a parallax scrolling effect vs. fixed
- Fully functioning navigation bar that can toggle and change focus corresponding with which part of the page you are on
- Display responsive images that has an overlay with text when you hover and can be clickable which takes you to another designated webpage
- Used font awesome icons to link all different social media accounts so that it takes you directly to their social media on click
